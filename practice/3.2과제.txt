문제 12. n^2 이 3의 배수이면 n은 3의 배수임을 증명하여라.
   1. 대우로 변경
   -> n이 3의 배수가 아니면 n^2 3의 배수가 아님을 증명
   2. 경우의 수
       1) n -> 3k+1 일때
	n^2 = (3k+1)^2 = 9k^2 + 6k + 1 = 3(3k^2 + 2k) + 1
	3(3k^2 + 2k) -> 3의 배수이고, 식에 +1이 남기 때문에 3의 배수가 아니다.
       2) n -> 3k+2 일때
	n^2 = (3k+2)^2 = 9k^2 + 12k + 1 = 3(3k^2 + 4k) + 1
	3(3k^2 + 4k) -> 3의 배수이고, 식에 +1이 남기 때문에 3의 배수가 아니다.
문제의 대우가 True 이기 때문에 n^2 이 3의 배수이면 n은 3의 배수이다.
